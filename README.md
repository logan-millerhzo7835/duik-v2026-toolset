## Overview

Duik is intended for motion designers and character animators who work in Adobe After Effects. It is designed to reduce the amount of manual setup involved in rigging and animation, while keeping the workflow organized and easier to manage.

At its core, the toolset provides 2D rigging capabilities, including IK systems, controllers, bones, keyframe operations, interpolation tools, and camera controls. It also exposes an API that can be used to extend Duik or tie it into other tools within an ExtendScript-based pipeline.

- Rigging tools for 2D animation in Adobe After Effects
